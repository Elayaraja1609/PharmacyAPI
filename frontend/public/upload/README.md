# Pharmacy Shop Images

Place your pharmacy shop photos in this folder.

## Required Images:
- pharmacy1.jpg
- pharmacy2.jpg
- pharmacy3.jpg
- pharmacy4.jpg
- pharmacy5.jpg

## Image Recommendations:
- **Size**: 1920x1080px (or similar 16:9 aspect ratio)
- **Format**: JPG or PNG
- **File Size**: Optimize images to keep file size reasonable (< 500KB each recommended)
- **Content**: Photos of your pharmacy shop interior, exterior, products, staff, etc.

## How to Add Images:
1. Place your images in this `frontend/public/upload/` folder
2. Name them exactly as: `pharmacy1.jpg`, `pharmacy2.jpg`, etc.
3. The slider will automatically display them

## Note:
If images are not found, a placeholder will be shown with instructions.
